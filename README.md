# Spring Boot REST API Project

## Overview
A RESTful CRUD API developed using Spring Boot, Spring Data JPA, MySQL, and Maven for managing student records.

## Features
- Create Student
- Retrieve Student Details
- Update Student Information
- Delete Student Records
- MySQL Database Integration

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

## API Endpoints

GET /students

GET /students/{id}

POST /student/add

PUT /student/update/{id}

DELETE /student/delete/{id}

## Database Setup

Run the provided `studentdb.sql` script to create the database and table.
